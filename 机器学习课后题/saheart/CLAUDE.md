## Project Overview

This repository contains the **SAheart dataset** — a retrospective sample of coronary heart disease (CHD) risk factors from a cardiovascular study in South Africa (n=462).

## Dataset: SAheart.csv

| Column     | Description                                  | Type    |
|------------|----------------------------------------------|---------|
| sbp        | Systolic blood pressure                      | numeric |
| tobacco    | Cumulative tobacco consumption (kg)          | numeric |
| ldl        | Low-density lipoprotein cholesterol          | numeric |
| adiposity  | Adiposity index                              | numeric |
| famhist    | Family history of heart disease              | categorical (`Present`/`Absent`) |
| typea      | Type-A behavior score                        | numeric |
| obesity    | Obesity score                                | numeric |
| alcohol    | Current alcohol consumption                  | numeric |
| age        | Age at onset                                 | numeric |
| **chd**    | **Response variable** — coronary heart disease | binary (0/1) |

There is no code, build system, or test suite in this repository. It is a data-only project.
